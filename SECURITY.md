# Security Policy

## Supported Versions

IPCalc GUI is currently in an early release stage. Security fixes are prioritized for the latest code and the latest released version.

| Version | Security support |
| --- | --- |
| Latest version | Supported |
| Older versions | Best effort |

## Reporting a Vulnerability

Please do not disclose exploitable security details in a public issue.

Prefer GitHub Security Advisories for private reporting. If the repository has not enabled that feature, open a public issue without sensitive details and ask to contact the maintainer privately.

Useful report details include:

- Affected version or commit.
- Reproduction environment.
- Impact.
- Minimal reproduction steps.
- Suggested fix, if available.

After a report is confirmed, maintainers will assess the impact, prepare a fix, and publish release notes when appropriate.
