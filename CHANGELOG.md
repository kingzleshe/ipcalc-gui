# Changelog

This file records notable changes to IPCalc GUI.

## Unreleased

## 0.2.0 - 2026-05-05

- Add open source project documentation, license metadata, and contribution guidelines.
- Add UI scale switching with persisted 100%, 110%, and 120% settings.
- Add IPv6 address list generation from forward and backward offsets.
- Improve IPv6 output labels and normalized input address display.
- Add a guard against excessively large generated address lists.

## 0.1.0

- Initial version.
- Add IPv4 and IPv6 network calculation.
- Support IPv4 CIDR, subnet mask, and wildcard mask input.
- Support IPv6 CIDR input.
- Add address range list generation.
- Add a native Slint desktop interface with theme switching.
