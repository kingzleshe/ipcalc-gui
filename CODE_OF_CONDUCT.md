# Code of Conduct

IPCalc GUI aims to keep project collaboration respectful, direct, and focused on facts. Please follow these principles when participating in issues, discussions, and pull requests.

## Expected Behavior

- Discuss specific problems with reproducible details and technical reasoning.
- Respect contributors with different backgrounds and experience levels.
- Accept maintainer decisions about project scope, complexity, and priority.
- Help clarify, reproduce, and verify issues when possible.

## Unacceptable Behavior

- Personal attacks, harassment, discriminatory language, or persistently hostile communication.
- Unrelated advertising, spam, or malicious links.
- Public disclosure of full exploit details for unresolved security issues.
- Intentional disruption of the project, tests, releases, or collaboration process.

Maintainers may remove inappropriate content, close unproductive discussions, or limit participation from people who violate this code of conduct.
